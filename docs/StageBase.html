Class: StageBase

StageBase(prefix, name, requires, produces, input, output)

Store information about a stage in a pipeline Derived classes must provide `run(runner, dataframe)` and `toJSON()`.

new StageBase(prefix, name, requires, produces, input, output)

Name Type Description
