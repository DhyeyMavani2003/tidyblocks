Class: DataFrame

DataFrame(values, oldColumns)

Store a dataframe. The frame is represented as an array of 0 or more rows, each of which is an object with exactly the same keys. The type of data associated with each key must be consistent for all rows. The column names are stored as well to handle the case where filtering reduces the dataframe to zero rows: it was confusing to have all the columns suddenly disappear.
