    <h3>Description:</h3>
    <p>This block allows for mathematical computations between two variables. The block accepts columns, numbers, and nested mathematical blocks as arguments. Multiple arithmetic blocks will follow the PEMDAS order of operations. </p>
    <h3>Arguments</h3>
    <p><b>Left Space:</b> This can be a column block, number block, or arithmetic block.</p>
    <p><b>Drop Down:</b> You can select addition, subtraction, multiplication, division, percentage, or power from the drop down menu.</p>
    <p><b>Right Space:</b> This can be a column block, number block, or arithmetic block.</p>
